Primeiros passos com Arduino

Aprenda Arduino usando 4 métodos diferentes:

  • Tradicional (Placa Arduino + Software Arduino)
  • Simulador (O aluno aprende Arduino mesmo sem ter a placa)
  • Fritzing (Placa Arduino + Software Fritzing) "profissionalize os projetos"
  • Scratch (Placa Arduino + Software Scratch) "ideal para crianças"

O que é Arduino?

O vídeo abaixo, apresentado por Massimo Banzi(criador do Arduino) conta um pouco da história e traz alguns exemplos do que pode ser feito com o Arduino. Ative as legendas.

Aula 1 - Primeiros passos com Arduino

Nesta aula nós vamos aprender a instalar e configurar o Arduino e também montar um pequeno circuito elétrico para acionamento de um Led.

Link para obter o Arduino:

www.arduino.cc

Aula 2 - Simulador de Arduino

Nesta aula vamos aprender a usar um simulador para desenvolver os projetos. O uso do simulador permite que o aluno faça os projetos mesmo não tendo a placa Arduino.

Acesso ao simulador:

circuits.io

Aula 3 - Arduino + Fritzing

Nesta aula nós vamos aprender a trabalhar com o Fritzing. O Fritzing permite modelar circuitos, confeccionar placas, shields e também programar o Arduino. O uso do Fritzing profissionaliza o desenvolvimento dos projetos.

Fritzing:

fritzing.org

Aula 4 - Arduino + Scratch

Usaremos a placa Arduino e o Software Scratch para desenvolver os projetos. O uso do Scratch permite que a programação do Arduino seja feita usando "blocos coloridos" (programação visual) e é ideal para crianças.

S4A:

s4a.cat
